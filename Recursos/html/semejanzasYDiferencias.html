Semejanzas entre XML y JSON


XML y JSON
Ambos son autodescriptivos (Legibles para las personas)
Su formato es texto plano
Jerárquico es decir sus valores pueden contener lista de objetos o valores
Muchos lenguajes de programación pueden analizar y utilizar tanto JSON como XML
Tanto JSON como XML se pueden obtener con XMLHttpRequest



Diferencias entre XML y JSON



XML JSON
Se considera un poco pesado y no tan ligero Más ligero y más rapido
Todos los valores son cadenas sin escribir y deben analizarse en tiempo de ejecución Utiliza objetos escritos
Hace uso de etiquetas al final No hace uso de una etiqueta al final
Es extenso Es más corto
No es muy rápido al leer y escribir Es más rápido de leer y escribir
No hace uso de arreglos Puede usar arreglos
